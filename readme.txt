Shortcode that allow your customers to SignIn with NEAR Wallet and make payments using NEAR token.

== Description ==

NEAR is a decentralized application platform which is built atop the NEAR Protocol, a revolutionary public proof-of-stake blockchain
which uses sharding to scale and an innovative account model to make apps similarly usable to those on todayâ€™s web.

Plugin allow you to switch network (mainnet/testnet) to test payment functionality on testnet and then take real NEAR tokens.

Don't have NEAR wallet? You can create it in few minutes:
- For mainnet: [https://wallet.near.org/](https://wallet.near.org/create "Create mainnet wallet")
- For testnet: [https://wallet.testnet.near.org/](https://wallet.testnet.near.org/create "Create testnet wallet")
